# 智能记帐程式 - 使用说明## 📋系统概览

这是一个完全在浏览器中运行的记帐系统，使用 localStorage储存所有资料，无需伺服器或资料库。所有资料都会保存在您的设备中。

## 🚀 快速开始

### 1.首次使用
- 直接在浏览器打开 HTML 档案
- 系统会自动填入当前日期和时间
- 开始记帐即可

### 2. 基本流程

## 📝 功能详解

### 1. 记帐功能（主页面）

**操作步骤：**
1. 选择或输入日期、时间（自动预填当前时间）
2. 选择收支别（收入/支出）
3. 选择或新增科目
4. 选择或新增地點
5. 输入金额
6. 填写收支摘要（可选）
7.拍照或上传收据（可选）
8. 点击「保存记录」

**收据功能：**
- 可直接拍照（手机）或上传图片
- 图片大小限制：2MB
- 支持预览和放大查看
-保存后可在记录列表中查看

### 2. 记录列表

**筛选功能：**
- 年份筛选
- 月份筛选
- 摘要关键字搜索
- 收支别筛选
- 科目筛选
- 地点筛选

**记录卡片：**
- 显示完整资讯
- 收入显示绿色边框
- 支出显示红色边框
- 可查看收据
- 可删除记录

### 3. 科目管理

**操作：**
1. 输入新科目名称
2. 点击「新增」
3. 系统会自动更新所有下拉选单

**删除限制：**
- 若有记录使用该科目，无法删除
- 需先删除或修改相关记录

### 4. 地点管理

**操作：**
1. 输入新地點名称
2. 点击「新增」
3. 系统会自动更新所有下拉选单

**删除限制：**
- 若有记录使用该地點，无法删除
- 需先删除或修改相关记录

### 5. 统计分析

**筛选统计：**
- 可选择特定年份/月份
- 显示该期间的总览、科目统计、地点统计

**总览数据：**
- 总收入
- 总支出
- 净收入（收入 - 支出）

**科目/地点统计：**
- 显示各项目的收入、支出、净额

### 6. 汇入/汇出（备份功能）

####📤 汇出资料

**三种方式（按浏览器支援度）：**

1. **选择路径汇出**（推荐）
   - 支援：Chrome 86+、Edge 86+
   - 可自订储存位置和档名
   - 档案名：`accounting_backup_YYYY-MM-DD.json`

2. **复制 JSON**
   - 所有浏览器支援
   - 显示 JSON 内容，可手动复制
   - 适合需要查看或修改内容的情况

3. **传统下载**（自动回退）
   - 下载到预设下载资料夹
   - 无法选择路径

#### 📥 汇入资料

**汇入模式：**

1. **合併模式**（预设）
   - 新增不重复的记录
   - 科目和地点自动去重
   - 不影响现有资料
   - 适用：跨设备同步、新增备份

2. **覆蓋模式**
   - 完全取代现有所有资料
   - **警告：会永久删除现有资料**
   - 适用：完整还原备份

**汇入流程：**
1. 选择 JSON 档案
2. 系统自动验证格式
3. 显示资料预览（包含统计）
4. 选择汇入模式
5. 确认后执行汇入
6. 自动更新所有界面

**JSON档案格式：**
```json
{
  "records": [ /* 所有记录 */ ],
  "categories": [ /* 所有科目 */ ],
  "locations": [ /* 所有地點 */ ],
  "exportDate": "2024-01-15T10:30:00.000Z",
  "version": "1.0"
}
